# 你不许调用 AI

在没有对应命令的情况下，阻止检测到 / 开头的消息时调用 AI

众所周知，“/”是 Astrbot 默认的唤醒词，如果你只是希望它检测命令，而不是调用 AI 时，你可能需要这个插件
